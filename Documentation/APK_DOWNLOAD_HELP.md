# Low Data APK Download Help

The release contains both a normal APK and a compressed APK archive.

```text
DesolationTheBackrooms.apk
DesolationTheBackrooms.apk.7z
DesolationTheBackroomsProject.zip
```

Android installs the `.apk` file directly.

Android does not install `.7z` directly. Use the `.7z` only when you need a smaller download:

1. Download `DesolationTheBackrooms.apk.7z`.
2. Extract it with an Android archive app that supports 7-Zip.
3. Install the extracted `DesolationTheBackrooms.apk`.
4. Delete the `.7z` after installing if storage is low.

Use the normal `.apk` when possible. Use the `.apk.7z` only for slow Wi-Fi or low-data sharing.

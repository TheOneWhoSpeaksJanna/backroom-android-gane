# True 3D Godot Implementation

This commit adds a real mesh-based Godot 4 Android version under `/godot`.

Implemented:

- A Godot 4 project.
- Real 3D mesh floors, ceilings, yellow walls, fluorescent fixtures, chairs, stains, player hazmat arms, and black entity geometry.
- First-person `CharacterBody3D` movement.
- Android touch joystick, right-side look, sprint, pause, settings, objective UI, death and win states.
- Procedural Level 0 layout.
- Three red breaker objectives and one strange exit door.
- A Godot Android GitHub Actions build workflow.

Expected workflow outputs:

```text
Builds/BackroomsLevelZero3D.apk
Source/BackroomsLevelZeroGodotProject.zip
```

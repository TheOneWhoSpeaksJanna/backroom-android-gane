# Backrooms Level Zero

**Backrooms Level Zero** is a playable first-person Android horror exploration game set only in Backrooms Level 0, also known as **The Lobby**. It focuses on liminal atmosphere: yellow wallpaper, dirty carpet, fluorescent ceiling lights, repeating office corridors, empty rooms, distant thuds, electrical buzzing, and psychological tension rather than jump-scare spam.

## Engine / renderer used

Unity and Godot are not available in this execution environment, so this repository uses a complete native Android implementation:

- Native Android Java
- OpenGL ES 2.0 renderer
- Android Canvas HUD/touch controls
- Procedurally generated geometry, textures, lighting effects, haze, vignette, film grain, and audio
- Android Gradle Plugin 8.7.3

This keeps the project fully buildable with standard Android tooling and avoids paid or copyrighted assets.

## Android requirements

- Package name: `com.backrooms.levelzero`
- Game name: `Backrooms Level Zero`
- Orientation: Landscape
- Minimum Android version: Android 8.0 / API 26
- Target SDK: 35
- Graphics: OpenGL ES 2.0 compatible device
- Recommended: mid-range Android phone or newer

## APK

The CI workflow builds the APK at:

```text
/Builds/BackroomsLevelZero.apk
```

The APK is debug-signed unless you configure release signing.

## How to install the APK

1. Download `Builds/BackroomsLevelZero.apk` from the repository or from the GitHub Actions artifact named `BackroomsLevelZero-APK`.
2. Copy it to an Android device.
3. Enable installation from unknown sources for your file manager or browser.
4. Open the APK and install it.
5. Launch **Backrooms Level Zero**.

You can also install with ADB:

```bash
adb install -r Builds/BackroomsLevelZero.apk
```

## How to open the source project

Open the repository root in Android Studio. Android Studio should detect the Gradle project automatically.

The source project ZIP generated by CI is located at:

```text
/Source/BackroomsLevelZeroProject.zip
```

## How to rebuild the APK

From the repository root:

```bash
gradle :app:assembleDebug
mkdir -p Builds
cp app/build/outputs/apk/debug/app-debug.apk Builds/BackroomsLevelZero.apk
```

The GitHub Actions workflow does this automatically on pushes to `main` and on manual `workflow_dispatch` runs.

## What is included

- First-person controller
- Left virtual joystick movement
- Right-side swipe camera look
- Sprint button
- Pause menu
- Sensitivity setting
- Graphics quality setting
- Exit game button
- Objective: find the strange exit door
- Procedural modular Level 0 layout
- Generated yellow wallpaper, carpet, ceiling tile, fluorescent light, and door textures
- Generated ambient horror audio: fluorescent hum, electrical buzz, distant thuds, footstep-like carpet pulses, and metallic ticks
- Haze/fog, vignette, film grain, flickering fluorescents, and subtle head bob

## Known limitations

This is not a Unity URP or Godot project because neither engine is available here. It is a native Android/OpenGL ES game that fulfills the same playable APK goal with generated assets. The APK produced by CI is debug-signed by default; release signing requires adding your own signing configuration locally or through GitHub secrets.

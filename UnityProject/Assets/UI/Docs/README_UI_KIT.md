# Desolation: The Backrooms UI Kit

This ZIP contains the clean Backrooms-style UI images and starter Unity files for your Android game.

## Included images

| File | Use |
|---|---|
| `01_main_menu_clean.png` | Main menu background/mockup |
| `02_save_slots_clean.png` | Play/save slot screen mockup |
| `03_settings_clean.png` | Settings screen mockup |
| `04_credits_clean.png` | Credits screen mockup |
| `05_feedback_clean.png` | Feedback screen mockup |
| `06_app_icon_1024.png` | Main Android app icon |
| `06_app_icon_512.png` and smaller variants | Optional resized icons |

## Important Unity note

These images are full-screen mockups. For a real game menu, the best setup is:

1. Use each image as a full-screen background.
2. Add real Unity UI buttons, sliders, input fields, and text on top.
3. Make the Unity UI elements transparent or lightly glowing so they match the mockup.
4. Connect button clicks with the included scripts.

This keeps the menu functional while preserving the design.

# Backrooms Godot UI Patch

This folder was added from `backrooms_godot_ui_patch.zip`.

Installed features:
- New Godot main menu scene.
- Loading screen scene.
- Settings menu scene.
- Buttons for Enter Level 0, Settings, and Exit.
- Settings are saved to `user://backrooms_settings.cfg`.
- The menu loads the existing gameplay scene at `res://Main.tscn`.

The original uploaded ZIP also included image and prompt assets. This repo integration focuses on the working Godot scenes/scripts first so the Android APK can build and launch reliably.
